SLIDE 02 / 07 — 共感と原因
「やりたくない」
の本当の理由
🧠 前頭前野がまだ未発達
未就学〜低学年は感情のブレーキが弱い
😨 失敗への恐怖
「また怒られる」「ダメな自分を見せたくない」
💬 日常の声かけの積み重ね
何気ない一言が脳の回避行動を作っている
指摘・否定 → 自尊心が傷つく
→ 脳が回避を選択 → 「やりたくない」
📌 キャプション：子どもが運動を嫌いになるのは、子どもの問題じゃない。脳の発達段階上、仕方のないこと。でもその恐怖心を大きくするか小さくするかは大人の声かけ次第です。
SLIDE 03 / 07 — NGな声かけ
❌ その褒め方、
逆効果かも！
「90点だったね、すごい！」
一見完璧な褒め言葉…でも⬇️
💭「90点の自分 = すごい」
＝「取れない自分 = すごくない」

結果褒め → 失敗が怖くなる
→ チャレンジしなくなる
→ 「やりたくない！」😤
🔍 これを「硬直的マインドセット」と言います。
挑戦を避け、失敗をひどく恐れる状態。
📌 キャプション：「結果褒め」が硬直的マインドセットを育てます。子どもが失敗を恐れ、挑戦しなくなる原因の一つ。無意識にやっちゃってる人、多いんです💦 次で正解を紹介！
SLIDE 04 / 07 — OKな声かけ
✅ 明日から使える
「過程褒め」フレーズ
✅ 「あの時一緒に練習頑張ったのが
よかったんだね！」
✅ 「あそこで諦めなかったのが
すごかったよ！」
✅ 「この部分、先週よりずっと
上手くなってるじゃん！」
📌 2つのポイント
👉 できなかった部分には触れない
👉 できた部分の「理由」を言葉にする
👉 足し算式でポジティブを積み重ねる
📌 キャプション：「過程を褒める」＝足し算式でポジティブを積み重ねること。これだけで子どもの自尊心が育ち「またやってみたい！」が自然と生まれます🌱
SLIDE 05 / 07 — 応用編
困った行動への
正しい対処法
❌ NG： 構いすぎ・叱りすぎ
→ 注意を引く行動がさらに強化される
✅ 正解： 「意図的な無関心」
困った行動 → 一言だけ言って淡々とスルー
良い行動 → すかさずたっぷり褒める！
🚨 本当に叱る場面はここだけ
他人への危害 / 自分の大怪我リスク
→ 短時間（30秒）・端的に・一度だけ
📌 キャプション：「完全無視」はNG。「見ているけど関わらない」が基本スタンス。子どもは本能的に関わってもらうことを求めています。良い行動の時こそ全力で反応してあげましょう😊
SLIDE 06 / 07 — まとめ
子どもは言葉より
「関心と雰囲気」
を見ている
🎯 結果ではなく行動・過程を褒める
🔍 できた部分の理由を具体的に言葉にする
🎉 良い行動の時こそ全力で関わる
「チャレンジすること自体が
すごいことなんだ！」
この価値観を毎日の声かけで育てていく🌱
📌 キャプション：テクニックより大切なのは、子どもへの「まなざし」。失敗しても大丈夫、挑戦したことが素晴らしい——その価値観が伝わる声かけを積み重ねていきましょう。
