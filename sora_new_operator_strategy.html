URGENT / 今週末の面談に向けて

新指定管理予定業者攻略
「運営計画に組み込まれる」ための戦略

前回（2021/2022）の指定管理者選定で四電工に敗れた業者が、2027年4月開始の次期管理者として戻ってくる前提。
今週末の面談で、5年契約の運営計画にSORAを組み込ませる。

0. 重要な前提整理

ユーザー情報：新指定管理予定業者は「前回（2022年スタートの選定）で四電工に敗れた業者」。つまりこの業者は「今度こそ取りたい」という強い動機を持っており、勝てる運営計画の構築途中。
CORE INSIGHT
「今が最大のチャンス」
運営計画策定中＝ゼロから入り込める／ 業者は差別化要素を強く欲している／ 5年契約に組み込まれれば長期パートナーシップ確定／ 入札勝利の助力者として相手から感謝される立場になる。

目次

  1. 業者候補の絞り込み（最有力＝公益財団法人 高松市スポーツ協会）
  2. 新業者が「今欲しいもの」を分解する
  3. SORAだけが提供できる5つの組込可能プログラム
  4. 面談前夜に作るA4 1枚提案
  5. 面談での具体的な進め方（候補別シナリオ）
  6. 「組み込まれた」と判断できるシグナル

2. 新業者が「今欲しいもの」を分解する

「運営計画に組み込まれる」ためには、相手が欲しいものを正確に把握する必要があります。前提（前回敗北→次回奪取）から逆算した、業者の具体的なニーズ。

2-A. 新業者が次回入札で勝つために必要な要素

要素業者の悩みSORAが提供できる解決
差別化された運営計画 前回は四電工に負けた。同じ提案では再び負ける。「他社にない目玉企画」が必要。 オリンピアン代表 × 子ども育成 × 棒高跳び専門の三位一体プログラム。これは他業者には絶対真似できない。
稼働率を上げる施策 四電工時代のB評価。稼働率改善の具体策が要る。 火・金・土夜の定期利用＋月次イベントで稼働率15-20%押し上げ可能。具体数字を提示。
市民満足度の向上策 地域住民・利用団体に「変わった感」を見せる必要 「香川出身オリンピアン主導の新コンテンツ」というメディア発信できる物語。
子ども・ジュニア育成の具体プログラム 市の評価項目「子ども振興」での得点が必要 既に観音寺・高松で稼働中。すぐ転用可能なノウハウとプログラム。
ユニバーサル・パラスポーツ対応 施設はバリアフリーだが活用しきれていない SORAコーチ陣で対応可能（要研修）。市の評価で配点高い項目。
室内棒高跳び施設の活用 「全国唯一の遊休資産」と指摘される SORA代表（リオ五輪棒高跳び代表）が世界唯一の活用責任者になり得る。
メディア露出 市議会・市の評価で「話題性」が重要 オリンピアン取材枠・日経新聞既出（ECC関連）の実績。
逆転の発想
「SORAは新業者の入札勝利を助ける戦略パートナー」
業者がSORAに「お願いする」立場になる構造を作る。SORAを運営計画に組み込むことが業者の入札勝利確率を上げる前提なら、断る理由がなくなる。

3. SORAだけが提供できる5つの組込可能プログラム

「運営計画書」に文字として書き込まれるレベルの具体提案。今週末の面談で全部を投げる必要はないが、相手の反応を見て使い分ける。

PROGRAM 1 — 目玉企画
「屋島ポールヴォルトアカデミー」共同立ち上げ
概要
全国唯一の室内棒高跳び公認施設を活かした全国レベルの棒高跳び育成拠点。リオ五輪代表の荻田大樹が直接指導。
想定対象
四国・西日本全域の中高生・大学生（30名規模）／単発体験会で年間500名
運営区分
業者：施設提供／SORA：プログラム運営・指導・集客
業者の評価加点
「全国唯一の施設活用」「県外集客」「メディア露出」
市民への打ち出し
「屋島から世界へ」のシンボル企画
PROGRAM 2 — 定期イベント
「月例 屋島チャレンジデー」共同主催
概要
毎月第3土曜に開催する子ども向け陸上記録会（無料 or 500円）。50m走・立ち幅跳び・ボール投げの3種目で自己ベスト挑戦。
想定参加
1回あたり100-150名（年間1,500人の利用増）
業者の評価加点
「子ども振興」「稼働率向上」「市民満足度」
運営区分
業者：施設・運営・広報／SORA：指導・記録会運営
PROGRAM 3 — 教室事業
「屋島キッズ陸上教室」週次レギュラー
概要
火曜・金曜・土曜の定期教室を業者公式プログラムとして組み込み（現在SORAの実質運営）
運営区分
SORAが運営委託を受ける形（業者からの委託料 or 参加費収入のレベニューシェア）
業者の評価加点
「定期利用」「子ども事業」「継続性」
既存活動の正式化
SORAは既に屋島で活動中。これを業者公式プログラムに「格上げ」する。
PROGRAM 4 — 年次フラッグシップ
「屋島スポーツフェスタ」年1回大型イベント
概要
年1回・1日500-1,000人規模の総合スポーツ体験イベント。オリンピアン招聘・地元企業協賛で運営。
予算規模
200-400万円（協賛金・参加費・市補助でカバー）
業者の評価加点
「話題性」「メディア露出」「年次の継続実績」
SORAの特出し
SUPERIOR・ECCかけっこの全国ネットワークで広域集客
PROGRAM 5 — ユニバーサル対応
「ユニバーサルスポーツ教室」月1回開催
概要
パラスポーツ・特別支援学校児童向けの月1回教室。施設のバリアフリー機能を活かす。
補助金
香川県・高松市の障がい者スポーツ振興補助、企業版ふるさと納税
業者の評価加点
「ユニバーサル対応」「DE&I」「地域共生」（評価項目で配点高い）
5本の組み立て方：1〜5すべてを最初から提示すると圧倒される。面談では「プログラム1（目玉企画）」と「プログラム3（既存活動の正式化）」の2本を中心に話し、他は「他にもこんなのが提案可能です」と一覧で渡す。

4. 面談前夜に作る「A4 1枚提案」

面談で手渡せるA4 1枚（または2枚）の提案メモを準備。業者がそれを持ち帰って社内で議論できる形にする。

📄 A4 1枚提案メモ（雛型）

タイトル案

「屋島レクザムフィールド 次期指定管理運営計画への協力提案 — Basic Sports Club SORA / 株式会社MOON」

構成（A4 1枚に収める）

  1. SORA概要（3行）：観音寺・高松で運営中の子ども向け陸上クラブ。代表は荻田大樹（リオ五輪 棒高跳び日本代表）。
  2. 屋島レクザムFとの関係（2行）：火・金・土の夜に高松スクールとして定期利用中。
  3. 協力提案の3本柱（各2-3行）：
    • ① 屋島ポールヴォルトアカデミー共同立ち上げ
    • ② 既存の子ども陸上教室の公式プログラム化
    • ③ 月例チャレンジデー＋年次スポーツフェスタ
  4. 業者にとってのメリット（5行）：
    • 差別化された運営計画（全国唯一の室内棒高跳び施設の活用）
    • 稼働率向上（年間1,500〜2,000人の追加利用）
    • 子ども事業の評価加点
    • メディア露出（オリンピアン取材吸引力）
    • SUPERIOR・ECC等の全国ネットワークによる広域集客
  5. 連絡先（1行）：株式会社MOON 代表 荻田大樹
渡し方：面談の中盤〜後半で「もしご興味あれば、こちらに簡単にまとめております」と自然に手渡し。その場で読ませない。「お時間あるときにご覧ください」で持ち帰ってもらう。

5. 面談での具体的な進め方（候補別シナリオ）

候補1：公益財団法人 高松市スポーツ協会 だった場合

特徴：公益財団法人＝市の外郭団体的存在。スポーツ事業の専門家。「市民満足度」「子ども事業」「ジュニア育成」が文化的に重視される。
「協会さんは既に高松市内のスポーツ施設をほぼ全部運営されていますよね。屋島レクザムFを取られたら事実上の完全制覇になりますね。」
「うちは香川出身の元日本代表として、長年地元のスポーツ育成に関わってきました。協会さんがやられているスポーツ少年団・健康体力つくり・コミュニティスポーツのいずれの専門部とも、自然に組める内容を持っています。」
「特に競技スポーツ専門部のジュニア育成と、屋島の室内棒高跳び施設の組み合わせは、協会さんならではの強い差別化になると思っているんです。」

協会向け 提案の重点

  • 「ジュニア育成のパイプライン」 — スポーツ少年団 → SORA → 全国 → オリンピック
  • 「他施設との連携」 — 協会の他33施設との横展開（例：総合体育館で冬の屋内練習）
  • 「公益性の高さ」 — ユニバーサルスポーツ・無料体験会など、公益財団法人らしい運営
  • 「市議会への報告ネタ」 — 全国唯一の施設活用は市議会で評価されやすい

候補2：シンコースポーツ・四電ビジネスグループ だった場合

特徴：民間と四電グループの共同体。商業ベース。「収益性」「サービス品質」「全国ノウハウ」が重視される。
「シンコーさんは全国でスポーツ施設を運営されていますよね。屋島は全国唯一の室内棒高跳び施設という独自性があるので、ここをブランド化できれば全国モデルになります。」
「うちの代表は元日本代表で、ECCかけっこスクールという全国事業の監修もしています。収益化できるコンテンツ提供は得意です。」

候補3：民間運営会社 だった場合

特徴：地縁が弱い。地元との関係構築を重視。「地元色のあるパートナー」を欲している。
「全国展開されている御社が香川に入られるとき、地元のアイコンになる元アスリートと組まれると、住民の受容が早いですよね。」
「うちは観音寺・高松で既に運営しているので、地元の保護者・学校・行政との既存関係をそのまま御社の運営に活かせます。」

候補不明（聞きながら判断）の場合

共通の初手：業者の「強み」と「これから取り組みたいこと」を聞く質問から入る。
「指定管理を取られた後、特に注力されたい領域はどんなことですか？」
「ご運営される他の施設では、どんな取り組みを大切にされていますか？」
「次回の指定管理の運営計画では、前回と比べて特に強化されたい部分はありますか？」

6. 「組み込まれた」と判断できるシグナル

面談中・面談後に、これらのシグナルが出れば「運営計画への組み込み」が現実的になっている証拠。

面談中の良いシグナル

  • 業者側から「具体的にどんな運営費負担が必要ですか」と金銭面の質問が出る
  • 業者側から「市にどうアピールできますか」と市への提案視点が出る
  • 「次回の応募書類に書ける形にまとめられますか」
  • 「うちの社内（評議会・理事会）に説明できる資料を頂けますか」
  • 「いつから動けますか」「2027年4月から間に合いますか」

面談後の良いシグナル

  • 3日以内にお礼メール＋次回面談の打診が来る
  • 「上司・理事と相談したので、こちらから連絡します」
  • 業者側の他の担当者（理事・専門部長など）を紹介される
  • 「市のスポーツ振興課に一緒に挨拶しませんか」

赤信号（撤退・別ルート検討）

  • 「うちは公平に対応するので特定団体だけ優遇は…」
  • 「指定管理を取れるか分からないので…」
  • 「すでに運営計画は固まっていて…」
  • 面談後の返答が1週間以上ない

面談後48時間以内の必須アクション

当日中：お礼メール送付

面談内容のサマリ＋特に話が盛り上がったポイントの言及＋次回面談打診

翌日：A4 1枚提案のブラッシュアップ版を送付

面談で出た相手のニーズを反映した改訂版。具体性を増す。

3日以内：相手の社内検討タイムラインを確認

「いつ頃お返事いただけそうですか」と直接聞く。曖昧なら追加情報提示

1週間以内：内部で「組み込まれる」確度を判定

シグナル分析・次回面談セット・並行ルート（市スポーツ振興課への挨拶等）の検討

7. まとめ — 「組み込まれる」ための一言

FINAL TAKEAWAY
「SORAを入れた運営計画は、入れない運営計画より勝つ確率が高い」
この単純な命題を、相手の頭の中に植え付ければ勝ち。
「今週末の面談は、5年契約の入り口。
1時間で全部決まる必要はない。だが、『この人と組むと勝てそうだ』という
一つの印象を相手の中に作れれば十分。」

面談で絶対に伝えたいワンライン（候補）

  • 「屋島レクザムFは全国唯一の室内棒高跳び公認施設。これを活かせる指導者は、おそらく日本に数人しかいない。」
  • 「うちが入れば、次回の運営計画書に『他社が真似できない目玉企画』を書ける。」
  • 「すでに屋島で活動している既存利用団体としても、御社の運営にスムーズに移行できる。」
