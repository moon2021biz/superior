1. ケース別 コスト試算

CASE A
中古ビルを買って軽リノベ
¥2,000〜6,500万
初期投資（リノベ込）

築年数の古い既存ビル（4-5階建て・延床400-500㎡相当）を購入。軽く改修してSORA BASE として使う。

項目金額レンジ備考
築20〜30年ビル取得1,500〜3,000万屋島・古高松エリア相場
築10〜20年ビル取得3,000〜5,000万状態良好
軽リノベ（内装・水回り・配線）500〜1,500万用途変更が伴うと増額
仲介手数料・登記・税金200〜400万取得価格の6-8%
初期合計2,200〜6,900万―
メリット
  • 新築の1/3〜1/2のコスト
  • 工期が短い（即入居可能）
  • 固定資産税が安い
  • キャッシュフロー余裕が出る
デメリット・リスク
  • 耐震性の確認必須（旧耐震だと致命）
  • 大規模修繕の費用が見えにくい
  • 用途変更（事務所→集会施設等）の法的制約
  • 屋島中町〜西町に出物が常にあるとは限らない

土地（170-250㎡）を購入し、新築で4-5階建てビル（延床400-500㎡）を建てる。長期保有・将来資産化を狙う。

設計・監理費建築費の8-10%1,300〜2,000万
各種許認可・解体・地盤改良200〜500万必要に応じて
登記・税金・諸経費300〜500万―
初期合計（S造ベース）約 1.5〜2.0億鉄骨造で見積もり

メリット
  • 用途・設計が自由（合宿対応・宿泊棟も組込可）
  • 新築なので耐震性・耐久性が高い
  • 融資が組みやすい（担保価値高い）
  • 長期保有で資産化／相続にも有利
デメリット・リスク
  • 初期投資が1.5-2.5億円と大きい
  • 計画から完成まで1.5-2年
  • 金利上昇局面で融資負担が増える
  • 地方の収益不動産は出口（売却）が難しい

3. 推奨ステップ

Phase 1（2027年4月〜）：CASE D テナント賃借から始める

初期費用 600〜1,700万円。これなら標準パターンの500万円調達枠内で（家賃前払い分まで含めて）スタートできる。失敗しても撤退コストが低い。

  • 立地：屋島レクザムFから徒歩・自転車圏（半径2km以内）
  • 規模：100〜200㎡の事務所兼ミーティングスペース（4-5階建て全てはまだ要らない）
  • 用途：会員受付・コーチ会議・面談・親御さん待機・座学・トレ機材保管

Phase 2（2028〜2029年）：CASE A or E に切替（実績を見て）

テナントで2-3年実績を作って、本気の拠点化を検討。中古ビル購入＋リノベ（A or E）でも3,000〜6,000万。融資・補助金で組める範囲。

  • 会員数の手応えがあれば物件購入の融資を組みやすい
  • クラウドファンディング（READYFOR等）併用で資金圧縮
  • 地域貢献ストーリー（空き家活用・地方創生）を活かして補助金獲得

Phase 3（2030年〜）：CASE B 新築でフラッグシップ化（任意）

BASE構想の本格化＝宿泊棟・食堂・トレーニングスペースを完備した拠点。投資1.5〜2.5億円は融資＋補助金＋クラウドファンディング＋スポンサーの多層調達で。

  • 事業実績10年・年商1億円超なら融資1.5億円は射程
  • 「アスリート育成のフラッグシップ」ストーリーで企業版ふるさと納税を狙う
  • 地元自治体（高松市）との包括連携協定を併用
注意：Phase 1からいきなりCASE B（1.5-2.5億）を狙うのは致命的なキャッシュフローリスク。前回のリアリティチェックで指摘した「補助金不採択」「学校等枠不適用」「YPV市場狭小」が同時発生すると、新築ビルの返済が事業を圧迫する。
「小さく始めて、実績ができてから物件化」が現実的な順序。

下見でチェックする5点

  1. 耐震基準：新耐震（1981年6月以降）かどうか必ず確認
  2. 用途地域・建築可否：「集会施設」「スポーツ施設」として使用可能か（市役所都市計画課で確認）

  4. 駐車場：SORA BASE なら最低5-10台欲しい（保護者送迎含む）
